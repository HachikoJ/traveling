# 五一广西·汕尾自驾旅行攻略

> 2026年4月28日 – 5月5日 · 深圳↔广西崇左靖西↔惠州汕尾 · 一家四口自驾旅行

## 功能

- 📊 **时间线总览** — 甘特图展示每天行程分布
- 📋 **逐日攻略** — 每天行程详情、提示、风险警告，支持折叠
- ✅ **完成打卡** — 勾选已完成项，手动输入实际完成时间
- ⏱ **智能调整** — 小偏差(<15min)只调下一项，大偏差(≥15min)自动调整后续
- 🔒 **固定时间** — 已购票的火车时间不会被调整
- ↩ **撤回** — 支持多步撤回
- 🎒 **行李清单** — 分类打包清单，勾选持久化
- 🚨 **意外预案** — 7个常见意外场景应对方案
- 📍 **今日高亮** — 旅行当天自动高亮当天行程
- 📱 **移动端适配**

```bash
# 方式2: Nginx
# 把三个文件放到网站根目录

# 方式3: GitHub Pages / Vercel / Netlify
# 直接推仓库，开 Pages 即可
```

## 技术栈

- 原生 HTML + CSS (Tailwind CDN) + JavaScript
- 零依赖，纯前端
- localStorage 持久化状态

## 文件结构

```
travel-plan.html   # 页面入口
travel-data.js     # 行程数据（8天60项）
travel-app.js      # 交互逻辑（打卡、调整、撤回等）
```
